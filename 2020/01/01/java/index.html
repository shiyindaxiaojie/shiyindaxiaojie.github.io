Java 基本特性

### Java 数据类型
Java 的数据类型分为基本类型和引用类型。
基本类型在声明时系统会自动给它分配空间，而引用类型声明时只是分配了引用空间，必须通过实例化开辟数据空间之后才可以赋值。例如，数组对象也是引用类型，将数组复制到另外一个数组只是复制了一个引用，对另外一个数组的修改对原来的数组也是可见的。
> Java 虚拟机没有提供 boolean 值专用的字节码指令，boolean 在编译之后都使用 int 基本类型来代替，占 32 位，而 boolean 数组将会被编码成 byte 数组，每个 boolean 元素占 8 位。

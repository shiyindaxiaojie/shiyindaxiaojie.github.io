安装目录结构

目录/文件 说明
bin/ 存放运行脚本和工具脚本，如 mysqld 客户端服务和 mysql 客户端连接工具
data/ 通过 msi 方式安装会自动创建这个目录，用于存放数据，每创建一个数据库在这个 data 目录自动生成数据库名的目录，如果是 innodb 引擎生成 *.frm 数据文件 和 *.ibd 索引文件，如果是 myisam 引擎，生成 *.myd 数据文件和 *.myi 索引文件
include/ C 和 C++ 语言实现 MySQL 存放的头文件
lib/ 类库文件
share/ 字符集
my.ini 数据库默认使用的配置文件
my-huge.ini 适合超大型数据库的配置文件
my-large.ini 适合大型数据库的配置文件
my-medium.ini 适合中型数据库的配置文件
my-small.ini 适合小型数据库的配置文件
my-template.ini 配置文件的模板，MySQL 配置向导将该配置文件中选择项写入到 my.ini 文件
my-innodb-heavy-4G.ini 表示该配置文件只对于 InnoDB 存储引擎有效，而且服务器的内存不能小于 4GB
